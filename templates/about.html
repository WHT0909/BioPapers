关于 BioPapers

什么是 BioPapers？

BioPapers（原名：PaperFinding 2.0）是一个基于 PubMed 的学术文献检索系统，旨在帮助生物医学领域的研究人员快速查找所需的学术论文。我们提供简洁直观的界面，让您轻松访问 PubMed 数据库中的海量文献资源。此外，BioPapers 内置了 DeepSeek 大模型，能够帮助用户分析文献、总结文章内容、分析创新点等

和 PaperFinding 相比做了哪些改进？

作为 PaperFinding 的 2.0 版本，我们改进了用户界面，并提供了更为全面的大模型助手功能。和第一版要求调用用户本地部署的模型相比，我们在 BioPapers 中允许用户直接调用官方 API 接口，使服务更加稳定（token 额度暂时由为爱发电的开发者提供😂）

基本功能

  • 关键词快速检索 - 支持布尔运算符
  • 显示文献详细信息（标题、作者、期刊、发表日期）
  • 提供文献摘要 - 帮助快速了解文献内容
  • DOI链接跳转 - 直接访问原文来源
  • 大模型互动问答 - 上传 PDF 或输入文本向大模型提问

使用说明

1. 在搜索框中输入您感兴趣的研究主题或关键词
2. 点击搜索按钮或按 Enter 键进行检索
3. 浏览搜索结果，点击文献标题查看详细信息
4. 使用底部分页按钮浏览更多结果
5. 在“文献助手”页面和 DeepSeek-V3.2 互动问答

技术细节

  • 后端: Python FastAPI
  • 前端: HTML5 + CSS3 + JavaScript
  • 数据库 API: NCBI Entrez E-utilities 官方 PubMed 接口
  • 大模型 API: DeepSeek-V3.2 非思考模式官方 API，上下文长度 128K，通过 LangChain 框架调用

注意事项

- 本系统仅提供文献检索服务，不存储或复制任何文献内容
- 文献版权归原作者和期刊所有
- 建议使用稳定的网络连接以获得最佳体验
- 如需获取全文，请通过 DOI 链接访问原文
