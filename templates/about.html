关于 PaperFinding

系统简介

PaperFinding 是一个基于PubMed的学术文献检索系统，旨在帮助研究人员和学生快速查找所需的学术论文。

数据来源

本系统使用 PubMed 数据库作为文献来源。PubMed是美国国家医学图书馆(NLM)维护的生物医学文献数据库，包含超过3500万篇生物医学文献。

技术架构

  • 后端: FastAPI (Python)
  • 前端: HTML5 + CSS3 + JavaScript
  • API: NCBI Entrez E-utilities

功能特点

  • 关键词快速检索
  • 分页浏览结果
  • 显示文献详细信息（标题、作者、期刊、发表日期）
  • 提供文献摘要
  • DOI链接跳转

使用说明

  1. 在搜索框中输入您感兴趣的研究主题或关键词
  2. 点击搜索按钮或按Enter键进行检索
  3. 浏览搜索结果，点击文献查看详细信息
  4. 使用底部分页按钮浏览更多结果
